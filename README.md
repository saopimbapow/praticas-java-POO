## 📚 Sobre o Repositório

Este repositório reúne implementações práticas dos principais pilares da POO em Java. Cada pasta representa um exercício ou mini-projeto independente, com foco em consolidar conceitos fundamentais da orientação a objetos.

---

## 🧩 Conceitos Abordados

```
📌 Herança                   → Reutilização e extensão de classes
📌 Encapsulamento            → Proteção e controle de atributos
📌 Polimorfismo              → Flexibilidade de comportamentos
📌 Abstração                 → Modelagem de conceitos do mundo real
📌 Modificadores de Acesso   → public, private, protected, default
📌 Bloco de Inicialização    → static { } e instância { }
📌 Enumerações (enum)        → Tipos de dados com valores fixos
📌 Classes e Objetos         → Base da orientação a objetos
```

## 🚀 Como Executar

### Pré-requisitos

- [Java JDK 11+](https://www.oracle.com/java/technologies/downloads/) instalado
- Um IDE de sua preferência (recomendo [IntelliJ IDEA](https://www.jetbrains.com/idea/) ou [VS Code](https://code.visualstudio.com/))

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/saopimbapow/praticas-java-POO.git

# 2. Acesse a pasta do exercício desejado
cd praticas-java-POO/Sistema_de_Funcionarios

# 3. Compile o arquivo Java
javac Main.java

# 4. Execute
java Main
```

> Cada pasta de exercício é independente. Navegue até o projeto desejado e execute conforme acima.

